TypeScript SDK

The TypeScript SDK lives at clients/typescript/skillhub/ and is published as @miqro/skillhub-client. It is not yet published to npm — reference it as a local file dependency.

Installation (local file dependency)

# In your frontend project
{
  "dependencies": {
    "@miqro/skillhub-client": "file:../clients/typescript/skillhub"
  }
}

# Build the SDK first
cd clients/typescript/skillhub
npm install
npm run build
Import policy: Import only from @miqro/skillhub-client. Internal paths (dist/domains/*, dist/types/*) are not part of the public compatibility contract.

Client initialization

import { SkillHubClient } from "@miqro/skillhub-client";

// Default: http://localhost:8080
const client = new SkillHubClient();

// Custom base URL (string form)
const client = new SkillHubClient("https://skillhub.example.com");

// Full options object (recommended)
const client = new SkillHubClient({
  baseUrl: "http://localhost:8080",
  credentials: "include",        // cookie-based session auth
  token: "sk_abc123",            // static bearer token
  getToken: () => localStorage.getItem("token") ?? undefined, // dynamic
  headers: { "X-Client": "web" },
  fetch: customFetch,            // SSR / test polyfill
});

Auth modes

Cookie session mode

const client = new SkillHubClient({
  baseUrl: "http://localhost:8080",
  credentials: "include",
});

const { data: user } = await client.login("username", "password");
// Cookie is set; subsequent requests use the session.

Bearer token mode

// Static token
const client = new SkillHubClient({
  baseUrl: "https://skillhub.example.com",
  token: "sk_abc123",
});

// Dynamic token
const client = new SkillHubClient({
  getToken: () => localStorage.getItem("skillhub_token") ?? undefined,
});

Error handling

Envelope mode (backward compatible)

const result = await client.search({ keyword: "agent" });
if (result.success && result.data) {
  console.log(result.data.skillIds);
} else {
  console.error(result.error?.code, result.error?.message);
}

Unwrap mode (throwing — recommended for page loaders)

import { SkillHubClient, SkillHubError } from "@miqro/skillhub-client";

try {
  const data = await client.unwrap(client.frontendSearch({ keyword: "agent" }));
  console.log(data.skillIds);
} catch (err) {
  if (err instanceof SkillHubError) {
    console.error(err.code, err.message, err.status);
    // err.code: "not_found", "unauthorized", "client.network_error"
    // err.status: HTTP status (404, 401, ...)
    // err.details: optional server-provided details
  }
}

Pagination iterators

Bounded async iterators for list endpoints. Default: max 10 pages. Always set maxPages explicitly. 

// Review queue
for await (const page of client.iterFrontendReviews({ size: 50, maxPages: 5 })) {
  for (const task of page.tasks) {
    console.log(task.skillName);
  }
}

// Search
for await (const page of client.iterFrontendSearch({
  keyword: "agent", maxPages: 3
})) {
  console.log(page.searchResult.skillIds);
}

// Community
for await (const page of client.iterFrontendIssues("ns", "slug", { maxPages: 3 })) {
  cacheIssues(page.issues);
}

// Releases
for await (const page of client.iterReleases("ns", "slug", { maxPages: 5 })) {
  console.log(page.releases);
}

Available iterators:

  • iterFrontendSearch(query)
  • iterFrontendReviews(options?)
  • iterFrontendPromotions(options?)
  • iterFrontendIssues(namespace, slug, options?)
  • iterFrontendDiscussions(namespace, slug, options?)
  • iterFrontendProposals(namespace, slug, options?)
  • iterReleases(namespace, slug, options?)
  • iterPipelineRuns(skillId, options?)

Logout

The SDK does not currently expose a public logout() method. Use a raw fetch call for now: 
await fetch(`${apiBase}/api/v1/auth/logout`, {
  method: "POST",
  credentials: "include",
});
// After logout: clear frontend auth state, redirect to /login

Community CRUD quick reference

// Issues
const { data: issues } = await client.listIssues("ns", "slug", { status: "OPEN" });
await client.createIssue("ns", "slug", { title: "Bug report", body: "..." });
await client.updateIssue("ns", "slug", 1, { status: "CLOSED" });
await client.addIssueComment("ns", "slug", 1, { body: "Fixed in v2." });

// Discussions
await client.createDiscussion("ns", "slug", {
  title: "Feature idea", category: "IDEAS", body: "..."
});

// Wiki
await client.createWikiPage("ns", "slug", {
  title: "Getting Started", slug: "getting-started", body: "# Hello"
});

// Proposals
await client.createProposal("ns", "slug", { title: "Refactor module X" });
